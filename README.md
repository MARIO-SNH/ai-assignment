This repository contains the implementation for the Artificial Intelligence assignment.

## Tasks Covered

### 1. Architecture Design
The architecture design for **Turing Test and CAPTCHA systems** is provided in the documentation.

Location:
docs/architecture.md

---

### 2. Simple Reflex Agent (AQI Agent)

A simple reflex agent is implemented that reads environmental sensor data from a file and calculates the **Air Quality Index (AQI)** for the region.

Files:
aqi_agent/aqi_agent.py  
aqi_agent/sample_sensor_data.csv  

Run the program:

python aqi_agent/aqi_agent.py --file aqi_agent/sample_sensor_data.csv

---

### 3. Uninformed Search Algorithms

The following search algorithms are implemented:

- Breadth First Search (BFS)
- Depth First Search (DFS)
- Depth Limited Search
- Iterative Deepening DFS

Location:
search/core.py

---

### Example Problem Implemented

Eight Queens Problem

Location:
problems/eight_queens.py

Run the search comparison:

python search/run_compare.py
